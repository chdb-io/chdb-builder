# chdb-builder
Experimental chdb docker builder for linux/x64

#### Features
- builds latest chdb from git
- all dependencies included
- ccache for fast rebuilds

### Usage
Run the included script to download, build and link a `chdb` python wheel

```
./build.sh
```
